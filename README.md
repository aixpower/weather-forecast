# codex
Coding environment for Codex

## Qweather Forecast Script

`qweather_forecast.py` retrieves the 7-day weather forecast for a given city using QWeather APIs. Usage:

```bash
python qweather_forecast.py <city_name>
```

Example:

```bash
python qweather_forecast.py Beijing
```

The script uses a fixed API key and host and prints a simple text forecast.
